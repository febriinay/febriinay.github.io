Pengalaman

Pengalaman saya dalam pembelajaran jarak jauh selama pandemi COVID-19 adalah saya menjadi tidak memiliki pengalaman apapun, saya hanya merasakan ke bosanan karena saya tidak bertemu dengan teman-teman secara langsung, belajarpun menjadi tidak kondusif, tugas saat pandemi lebih banyak dibandingkan sebelum pandemi.
